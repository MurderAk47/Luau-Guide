If / Else 🤔

Your code can make decisions. "If the player's health is above 0, they're alive. Otherwise, they're dead." That kind of logic is what makes games feel alive — and it all starts with if / else.

🔀 Basic if / else

The structure is: if something is true then do this, else do something else, end.

ifelse.lua
local health = 50
 
if health > 0 then
print("You are alive!")
else
print("You are dead!")
end
⚠️
Don't forget end! Every if-statement must be closed with end. Forgetting it is the #1 beginner mistake in Luau.

🔣 Comparison operators

These symbols go inside your if condition to compare things:

== is equal to
~= is NOT equal to
> greater than
< less than
>= greater than or equal
<= less than or equal
🚫
= vs == — Single = sets a value. Double == checks if two things are equal. Using the wrong one will either crash or give weird results!

⛓️ elseif — multiple conditions

You can check several conditions in a chain using elseif:

elseif.lua
local health = 30
 
if health > 75 then
print("Healthy!")
elseif health > 25 then
print("Low health!") -- ← this runs
else
print("Critical! Run!")
end

🔗 and / or — combining conditions

andor.lua
local health = 50
local hasShield = true
 
-- both must be true
if health > 0 and hasShield then
print("Alive with shield!")
end
 
-- at least one must be true
if health > 100 or hasShield then
print("Safe for now!")
end
