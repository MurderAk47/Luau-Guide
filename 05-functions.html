Functions 🛠️

A function is a named chunk of code you write once and can run any time just by calling its name. Instead of writing the same 10 lines over and over, you write them once in a function and call it whenever you need it. Functions are the backbone of every good script.

📝 Creating and calling a function

functions.lua
-- Define the function
local function sayHello()
print("Hello, Zack!")
end
 
-- Call (run) the function
sayHello() -- Hello, Zack!
sayHello() -- Hello, Zack! (again)
sayHello() -- Hello, Zack! (and again)

📥 Functions with parameters (inputs)

You can pass information into a function using parameters:

params.lua
local function greetPlayer(name, score)
print("Welcome, " .. name .. "!")
print("Your score: " .. score)
end
 
greetPlayer("Zack", 99)
greetPlayer("Alice", 45)
💡
Parameters are like variables that only exist inside the function. When you call the function, you provide the actual values (called arguments).

📤 Functions that return a value

Functions can send a value back out using return:

return.lua
local function addNumbers(a, b)
return a + b
end
 
local result = addNumbers(10, 5)
print(result) -- 15
 
local function isAlive(health)
return health > 0 -- returns true or false
end
 
print(isAlive(50)) -- true
print(isAlive(0)) -- false
