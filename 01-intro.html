What is Luau? 🎮

Luau is the programming language Roblox uses to make games do stuff. Doors that open, swords that deal damage, leaderboards — all of it is written in Luau. It's based on a language called Lua, but Roblox added extra features on top. Learn Luau and you basically learn Lua too.

💬 Your first line of code

The very first thing everyone writes is a print() statement. It sends a message to the Output window inside Roblox Studio. Think of it like a sticky note from your script.

script.lua
-- This is a comment. Luau ignores it completely. -- Use comments to leave notes for yourself! print("Hello Zack! Your script works!")
💡
Comments start with two dashes --. Everything after them on that line is ignored by Luau. Use them to explain your code!

📁 Where do you write scripts?

In Roblox Studio, scripts live inside objects in your game. There are 3 types you'll use:

📜
Script
Runs on the server. Everyone in the game sees the effects. Use for most game logic.
💻
LocalScript
Runs on one player's screen only. Use for GUIs, camera, and player input.
🧩
ModuleScript
A reusable toolbox. Holds shared code that other scripts can load and use.
⚠️
Classic beginner mistake: Putting a LocalScript where a regular Script should go. Your code will run but nothing visible will happen — super confusing. When in doubt, start with a Script.
