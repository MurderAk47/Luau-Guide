Loops 🔄

Imagine you need to spawn 100 enemies. Writing the same line 100 times is insane. That's where loops come in — they repeat code for you automatically. There are 3 loop types in Luau and each has its own job.

🔢 for loop — repeat a set number of times

A for loop counts from one number to another, running the code each time:

forloop.lua
for i = 1, 5 do
print("Count: " .. i)
end
 
-- Output:
-- Count: 1
-- Count: 2
-- Count: 3
-- Count: 4
-- Count: 5

You can also count backwards or by bigger steps:

forstep.lua
-- Count down from 10 to 1
for i = 10, 1, -1 do
print(i)
end
 
-- Count by 2s (2, 4, 6, 8, 10)
for i = 2, 10, 2 do
print(i)
end

🔁 while loop — repeat while something is true

A while loop keeps going as long as its condition is true. Perfect for things like "keep the game running while the player is alive".

whileloop.lua
local count = 0
 
while count < 3 do
count = count + 1
print("Count: " .. count)
end
 
print("Done!")
🚫
Infinite loops! If your condition never becomes false, the loop runs forever and crashes Roblox Studio. Always make sure something inside the loop can make the condition false!

⏱️ task.wait() — adding delays

Loops run instantly by default — thousands of times per second. Use task.wait() inside a loop to add a pause:

spawn.lua
-- Spawn a coin every 2 seconds, 5 times
for i = 1, 5 do
print("Spawning coin " .. i)
task.wait(2) -- pause 2 seconds
end
print("All coins spawned!")
✅
Always use task.wait() instead of the old wait(). The task version is more accurate and is the modern Roblox standard.
